Método de Gauss

Este método realiza la eliminación hacia adelante para obtener una matriz triangular superior. No aplica retro-sustitución ni reducción completa.

Sistema de ecuaciones (opcional)

Escribe un sistema lineal y luego usa “Cargar sistema en matriz” para que se llene la matriz aumentada automáticamente.

Una ecuación por línea, siempre con =. Puedes usar multiplicación implícita como 2x, 3(x+1), etc.

Si lo dejas vacío, el programa detecta las variables y las ordena alfabéticamente.

Este botón rellenará la matriz aumentada con los coeficientes y el término independiente b según el sistema escrito.
